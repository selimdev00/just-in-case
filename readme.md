# Just in case

Admin panel with backend 

## Tech

- Node.js
- Express.js
- Nuxt.js
- TailwindCSS
- PrimeVue
- NuxtAuth

## Client

Run client with `npm run dev`

## Database & server

Run migrations with `npx sequelize-cli db:migrate`

Run seeders with `npx sequelize-cli db:seed:all`

Run server with `npm run dev`

Run tests with `npm run test`
